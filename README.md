> A powerful beautiful scrollbar based on Vue.js for PC and mobile.
> [vuescrolljs.yvescoding.org](http://vuescrolljs.yvescoding.org/)

## Features

- Support mobile and phone
- Support Smooth scroll
- Support customize scrollbar
- Support pull to refresh and push to load
- Support typescript
- ...
