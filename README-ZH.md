> 一款基于 Vue.js 的，支持 Pc 和移动端的功能强大的漂亮的滚动条。
> [vuescrolljs.yvescoding.org](http://vuescrolljs.yvescoding.org/zh)

## Features

- 支持 Pc 和移动端
- 支持平滑地滚动
- 支持自定义滚动条
- 支持下拉刷新和上推加载
- 支持 typescript
- ...
